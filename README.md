# pixel-sync

Пиксель-арт рандомайзер для дизайн-синков. Персонажи живут на маленькой ферме — кто-то готовит, кто-то рубит дрова, кто-то ловит рыбу. Нажимаешь кнопку — камера зумится на случайного, и он отвечает на твой вопрос.

Один файл (`index.html`), никакой сборки, ничего не нужно ставить.

## Запуск

```bash
git clone https://github.com/<your-user>/pixel-sync.git
cd pixel-sync
open index.html
```

Или просто двойной клик по `index.html` — откроется в браузере.

Тестировалось в Chrome / Safari / Firefox. Нужен интернет один раз, чтобы подгрузился пиксельный шрифт с Google Fonts (после кеша работает оффлайн).

## Управление

| Действие | Что делает |
|----------|------------|
| Кнопка **КТО ОТВЕЧАЕТ?** или `пробел` | Рулетка → камера зумится на случайного персонажа |
| Кнопка **ДАЛЬШЕ** или `esc` | Возврат к общему виду |
| Шестерёнка внизу справа | Открыть панель участников |
| `esc` при открытой панели | Закрыть панель |

## Редактирование участников

Шестерёнка внизу справа → панель с участниками. Там можно:

- **Переименовать** любого — просто отредактировать инпут, сохранится автоматически в localStorage
- **Удалить** (кроме последнего)
- **Добавить нового** — кнопка `+ ДОБАВИТЬ НОВОГО` спавнит случайного «бродягу» с рандомным цветом одежды и активностью (`листает фигму`, `наливает кофе`, и т.п.)

Бродяги в отличие от персонажей со станциями просто гуляют по карте — у них нет фиксированного места. Они тоже попадают в рандомайзер.

Все изменения хранятся в localStorage браузера. Сбросить к дефолтам — открыть консоль и:

```js
localStorage.removeItem('pixel-sync-v1'); location.reload();
```

## Что внутри сцены

- Большой дом с черепичной крышей (сверху-справа)
- Котедж с дымоходом (снизу-справа)
- Тёмный сарай (сверху-слева)
- Грядка обычная и грядка с забором
- Маленькая лужа с рыбаком
- Костёр, мольберт, пень с топором, лавочка под деревом, печь
- Лес плотной стеной по периметру, разбросаны мухоморы и цветы
- Четверо котов бродят у лужи (рыжий, белый, серый, чёрный)
- Бабочки

## Кастомизация

Базовые параметры — в начале `<script>` в `index.html`:

- `DEFAULT_CHARACTERS` — стартовый список (имя, станция, цвета одежды)
- `STATIONS` — где какие станции стоят на карте + координаты для персонажа
- `WANDER_ACTIVITIES` — пул активностей для бродяг
- `RND_COLORS` — палитры для случайной генерации
- `CATS` — массив котов с типами и стартовыми позициями

Палитра сцены (`const C = {...}`) — все цвета травы, дерева, черепицы, и т.д. — там же.
